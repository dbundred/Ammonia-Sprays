# Ammonia-Sprays
Case Setups for Ammonia Spray Simulations.

The repository contains the input and data files required to run the simulations included in the paper "Capturing transition behaviour of flash boiling ammonia sprays using a thermally controlled breakup model"

The repository also contains the udf shared object file, which will allow users to use the model developed in the paper. 

Please contact the authors for access to the udf souurce code.
